# Customer Churn Analytics & Prediction

## Project Overview
Customer Churn Analytics & Prediction is a data analytics and machine learning project that analyzes customer behavior and identifies customers likely to leave a company.

## Business Problem
Customer churn directly affects company revenue. Businesses need to understand why customers leave and which customer groups are at higher risk.

This project answers:
- What is the overall churn rate?
- Which contract type has the highest churn?
- Which payment method is linked with churn?
- What is the revenue at risk?
- Which customers are high risk?

## Tools Used
- SQL
- Python
- Power BI
- Machine Learning
- GitHub

## Dataset
The dataset contains customer information such as:
- Customer ID
- Gender
- Senior citizen status
- Tenure months
- Contract type
- Payment method
- Monthly charges
- Total charges
- Churn status

## Key KPIs
- Total Customers
- Churned Customers
- Churn Rate %
- Monthly Revenue at Risk
- Churn by Contract Type
- Churn by Payment Method
- Churn by Tenure Group

## SQL Analysis
SQL queries were written to analyze churn count, churn rate, high-risk customers, revenue at risk, and churn patterns by customer attributes.

## ML Plan
A basic machine learning model can be used to predict whether a customer is likely to churn based on tenure, contract type, payment method, and charges.

## Expected Insights
- Month-to-month customers are more likely to churn.
- Electronic check users show higher churn risk.
- Customers with low tenure are more likely to leave.
- Higher monthly charges may increase churn risk.

## Project Outcome
This project demonstrates how data analytics and machine learning can help businesses reduce customer churn and improve customer retention.
